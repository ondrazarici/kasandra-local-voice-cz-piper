# Kasandra - Piper female voice (CZ)

Kasandra je vysoce kvalitní, lokální český hlasový model vytvořený pro **Piper TTS**. Je navržen tak, aby přinesl přirozeně znějící českou syntézu řeči do tvé chytré domácnosti (např. Home Assistant) bez nutnosti odesílat jediný bajt dat do cloudu.

## Proč Kasandra?
Většina hlasových asistentů buď neumí česky, nebo vyžaduje připojení k internetu a odesílání hlasu na servery velkých korporací. Kasandra to mění:
* **Kompletně offline:** Vše běží na tvém hardwaru.
* **Soukromí:** Žádná data neopouštějí tvou síť.
* **Nízká latence:** Rychlá odezva díky optimalizaci pro Piper.
* **Přirozený projev:** Optimalizováno pro srozumitelnou a lidsky znějící češtinu.

## Technické informace
* **Formát:** ONNX model (kompatibilní s Piper).
* **Jazyk:** Čeština (CZ).
* **Určení:** Ideální pro Home Assistant (přes doplněk Piper/Wyoming protocol).
